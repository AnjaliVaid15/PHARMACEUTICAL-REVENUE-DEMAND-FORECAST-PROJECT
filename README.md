 **Pharmaceutical Revenue Analysis & Demand Forecasting Dashboard**
 _Project Summary_

Developed an end-to-end data analytics project analyzing pharmaceutical revenue data to evaluate drug portfolio performance, identify growth trends, measure volatility, and generate demand forecasts.
The project demonstrates hands-on experience in data cleaning, SQL querying, data modeling, business intelligence, KPI development, and predictive analytics using Power BI.


**End-to-End Data Analytics Workflow**
1️⃣ Data Acquisition 
:Extracted structured dataset from Kaggle
Dataset included: Drug Category, Revenue, Year, Month, Quarter
:Imported raw CSV data for preprocessing

2️⃣ Data Cleaning & Transformation (Microsoft Excel)
:Performed data preprocessing and transformation:
:Removed duplicates and inconsistencies
:Handled missing values
:Standardized date formats
:Created derived columns (Year, Month, Quarter)
:Validated revenue data types
:Structured dataset for relational analysis

3️⃣ SQL Data Analysis & Business Logic Validation
:Loaded cleaned dataset into SQL database for analytical processing.
Implemented:
-Revenue aggregation by Year, Quarter, Drug Category
-Portfolio contribution analysis
-YoY Growth % calculation
-Identification of top-performing and underperforming products
-Volatility assessment using revenue fluctuations
-Average monthly revenue computation

SQL Concepts Used:
-GROUP BY, ORDER BY
-Aggregate Functions (SUM, AVG)

4️⃣ Data Visualization & Dashboard Development (Power BI)
:Designed an interactive Business Intelligence Dashboard to deliver executive-level insights.
Key Features:
-KPI Cards (YoY Growth %, Avg Monthly Revenue, Top Drug, Lowest Drug)
-Quarterly Revenue Contribution Analysis
-Monthly Revenue Trend Visualization
-Revenue Heatmap (Matrix with Conditional Formatting)
-YoY Growth Analysis (Combo Chart)
-Revenue Forecast using Time-Series Forecasting (Exponential Smoothing)

Implemented:
-DAX Measures
-Data Modeling
-Interactive Slicers
-Drill-down analysis
-Forecast with 95% Confidence Interval


**Key Business Insights**
1 YoY growth declined by 26%, indicating revenue contraction.
2 Revenue peaked between 2016–2018 before entering a correction phase.
3 Monthly trend reflects weakening demand momentum.
4 Q4 demonstrates strong seasonal performance.
5 Mid-year quarters show comparatively lower contribution.
